RefBoard 사용설명서

이미지를 무한 캔버스에 자유롭게 모으고, 정렬하고, 한 장으로 묶고, 링크로 공유하는 레퍼런스 보드. 이 문서는 실제 화면 구성을 그대로 재현해 기능을 안내합니다.

2이미지 넣기

네 가지 방법 중 편한 것을 쓰면 됩니다. 모두 여러 장을 한 번에 넣을 수 있습니다.

  • 드래그 앤 드롭 — 탐색기나 웹브라우저에서 이미지를 끌어다 캔버스에 놓습니다. 놓은 위치에 배치됩니다.
  • 붙여넣기 — 이미지를 복사한 뒤 Ctrl+V. 화면 중앙에 들어옵니다. (스크린샷 캡처 직후에 특히 편리)
  • 파일에서 가져오기 — Ctrl+I 또는 툴바의 이미지 가져오기 버튼으로 파일 선택창을 엽니다.
  • 데스크탑 앱 — OS 파일 드롭도 그대로 지원합니다.
웹 주소(URL)를 붙여넣어 이미지를 가져오는 기능은 현재 버전에는 없습니다. 이미지를 직접 복사해 Ctrl+V 하거나 파일로 가져오세요.

정렬 · 분배

  • 가장자리 정렬: Ctrl+← / → / ↑ / ↓ (왼·오른·위·아래)
  • 균등 분배: 수평 Ctrl+Shift+← · 수직 Ctrl+Shift+↑
  • 가운데 정렬(가로·세로)은 커맨드 팔레트나 툴바에서 실행

크기 맞추기

  • 폭 통일 Ctrl+Alt+↑ · 높이 통일 Ctrl+Alt+→ · 배율 통일 Ctrl+Alt+← · 면적 통일 Ctrl+Alt+↓ (기준은 첫 선택 항목)

자동 배치 · 격자

  • 자동 패킹 Ctrl+P — 모든 이미지를 빈틈없이 재배치
  • 캔버스 최적화 Ctrl+Alt+P — 패킹 후 전체 보기까지 한 번에
  • 격자 정렬 — 이름순 Ctrl+Alt+N · 추가순 Ctrl+Alt+A · 레이어순 Ctrl+Alt+O · 무작위 Ctrl+Alt+R (같은 키 반복 시 오름/내림 토글)

겹침 순서 · 그룹

  • 앞으로 ] · 맨 앞 Shift+] · 뒤로 [ · 맨 뒤 Shift+[
  • 그룹 Ctrl+G · 그룹 해제 Ctrl+Shift+G

8웹으로 공유하기

보드를 업로드해 링크로 공유합니다. 받는 사람은 설치 없이 브라우저에서 읽기 전용으로 봅니다.

  1. 툴바의 웹 링크 공유 또는 Ctrl+Shift+S (최초 1회 구글 로그인)
  2. 공개 범위·만료·허용 이메일을 정하고 업로드
  3. 생성된 뷰어 링크가 클립보드에 자동 복사 → 전달
보드 공유
🔒 비공개 — 허용한 사람만 (기본)
⏰ 만료 기간 — 설정 안 함
✉️ 허용 이메일 — alice@example.com
취소공유

공개 vs 비공개

설정누가 볼 수 있나
비공개 (기본)허용 이메일 목록에 있는 계정만 — 그 계정으로 로그인해야 보임
공개링크를 아는 누구나 (로그인 불필요)

상태바의 배지로 현재 상태를 확인할 수 있습니다: 로컬(미공유) · 공개 · 비공개. 공유 보드 목록은 공유 관리 패널에서 확인·전환·삭제할 수 있습니다.

기본은 비공개입니다. 공개하려면 공유 창에서 ‘공개’를 직접 켜야 해요. 공개로 두면 링크가 퍼질 경우 누구나 봅니다 — 민감한 보드는 비공개로 두고 허용 이메일을 추가하세요(받는 사람은 그 이메일로 로그인해야 열람).

10웹으로 쓰기 · 데스크탑 앱 설치

설치 없이 웹에서 바로 쓰거나, 데스크탑 앱으로 설치해 창을 띄워두고 쓸 수 있습니다.

  • 웹 (권장) — 브라우저에서 refboard-e0m.pages.dev 접속. 설치가 필요 없고 항상 최신입니다. 보드는 브라우저에 자동 저장되고 .refb 파일로도 보관할 수 있습니다.
  • 데스크탑 앱 — .msi 설치 파일로 Windows에 설치하면 항상 위·창 불투명도 같은 작업용 창 기능을 쓸 수 있습니다.
설치할 때 Windows 보안 경고가 뜹니다. 이 앱은 아직 코드 서명이 없어서 .msi를 실행하면 “Windows의 PC 보호”(SmartScreen)나 “알 수 없는 게시자” 경고가 나타납니다. 직접 받은 신뢰할 수 있는 파일이라면 추가 정보 → 실행을 눌러 설치하세요. 꺼림칙하면 설치 없이 웹으로 쓰는 것을 권합니다.

11알려진 한계와 피드백

이 버전은 알파(시험) 단계입니다. 기능과 화면이 예고 없이 바뀔 수 있고 드물게 버그가 있을 수 있어요. 중요한 보드는 Ctrl+S로 .refb 파일로도 따로 저장해 두세요.

현재 한계

  • 편집은 데스크탑(마우스) 권장 — 휴대폰·태블릿에서 공유 링크를 보는 것은 잘 되지만, 편집 조작(핸들·드래그)은 제한적입니다.
  • 한국어 화면만 제공합니다.
  • 비공개로 공유한 보드는 받는 사람도 구글 로그인이 필요합니다(공개 보드는 로그인 없이 열람).
  • 웹 주소(URL)로 이미지 가져오기는 아직 없습니다 — 복사 후 붙여넣기 또는 파일 가져오기를 쓰세요.
  • 데스크탑 앱 .msi는 코드 서명이 없어 설치 때 보안 경고가 납니다(위 10번 참고).
