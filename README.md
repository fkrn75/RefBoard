# RefBoard

> PureRef 호환 레퍼런스 이미지 보드 + 웹 공유 (개발 중)

무한 캔버스에 레퍼런스 이미지를 모아 자유롭게 배치·정리하는 데스크탑 도구입니다. [PureRef](https://www.pureref.com/)의 핵심 기능을 구현하고, 추가로 **보드를 웹 링크로 공유해 다른 PC·모바일에서도 열람**하는 기능을 목표로 합니다.

## 현재 상태 (MVP 개발 중)

- ✅ 무한 캔버스 (휠 줌 · 우클릭 팬)
- ✅ 이미지 가져오기 (드래그드롭 · 클립보드 붙여넣기)
- ✅ 선택 (단일 · Shift 다중 · 러버밴드) + 외곽선 표시
- ✅ 드래그 이동 (다중 동시)
- ⬜ 자동 패킹 · 삭제/복제 · 저장(.refb) — 진행 예정
- ⬜ Tauri 데스크탑 셸 (항상 위 등) · 웹 공유 — 이후 단계

상세 진행은 [체크리스트](체크리스트.md), 전체 기능 명세는 [기능명세서](기능명세서.md)를 참고하세요.

## 기술 스택

- **데스크탑**: Tauri 2 (예정) + Vite + TypeScript
- **렌더링**: PixiJS (WebGL) — 무한 캔버스 · 수백 장 이미지 대응
- **웹 공유**: `core-canvas`를 재사용한 읽기 전용 웹 뷰어 (예정)

핵심 설계: 렌더링을 웹 기술로 통일해 **데스크탑 앱과 웹 뷰어가 같은 엔진(`core-canvas`)을 공유**합니다.

## 개발

```bash
cd app
npm install
npm run dev        # http://localhost:1420
```
